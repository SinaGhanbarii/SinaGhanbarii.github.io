This project developed and compared classical machine-learning methods and neural networks for predicting thermodynamic properties of molecules.

## Work completed

- Prepared molecular feature sets based on quantum descriptors and functional groups
- Trained and compared multiple regression algorithms
- Developed neural-network models
- Evaluated model accuracy and generalisation
- Structured the workflow for reproducible analysis in Python

The project demonstrates how chemically meaningful descriptors can be combined with data-driven models for property estimation.
