The **gPROMS ModelBuilder VS Code Extension** provides a modern development environment for writing and editing gPROMS ModelBuilder and GPL code in Visual Studio Code.

## Main features

- Syntax highlighting for gPROMS language constructs
- More than 65 reusable code snippets
- Context-aware autocomplete
- Hover documentation
- Real-time diagnostics and error checking
- Code folding for models, processes, loops, and tasks
- Support for `.gproms` and `.gpl` files
- Dark and light editor themes

## Technical implementation

The extension is implemented in JavaScript using the Visual Studio Code Extension API and Language Server Protocol. It includes a TextMate grammar, language server, symbol parser, snippet library, and packaged VSIX installer.
