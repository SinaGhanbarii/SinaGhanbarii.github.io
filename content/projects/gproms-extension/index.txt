The **gPROMS ModelBuilder VS Code Extension** provides an improved development environment for writing and editing gPROMS ModelBuilder and GPL code directly in Visual Studio Code.

The project was developed to make process-model development faster, clearer, and less error-prone by introducing language-support features that are normally available in modern programming environments.

## Main features

* Syntax highlighting for gPROMS blocks, section headers, mathematical functions, control-flow statements, PDE operators, variables, numbers, strings, and comments
* More than 65 ready-to-use code snippets for models, processes, tasks, reactors, heat exchangers, optimisation, estimation, schedules, and distributed systems
* Context-aware autocomplete for declared variables, parameters, unit names, keywords, and built-in functions
* Hover documentation containing syntax descriptions and working examples
* Real-time error checking for missing semicolons, unclosed blocks, misspelled keywords, invalid assignment operators, and misplaced sections
* Block folding for models, processes, loops, tasks, sequences, and parallel blocks
* Automatic comment toggling and bracket handling
* Support for `.gproms` and `.gpl` files

## Technical implementation

The extension is implemented in JavaScript using the Visual Studio Code Extension API and Language Server Protocol.

It includes:

* A TextMate grammar for syntax highlighting
* A language server for diagnostics and hover documentation
* A symbol parser for context-aware autocomplete
* A snippet library for common process-modelling structures
* Custom dark and light colour configurations
* A packaged `.vsix` installer with no external runtime dependencies

## Motivation

gPROMS ModelBuilder is widely used for equation-oriented process modelling, dynamic simulation, optimisation, and parameter estimation. However, its code-editing workflow does not provide many of the language-assistance features commonly available in modern code editors.

This extension was created to provide a more efficient and accessible coding environment for students, researchers, and process-systems engineers working with gPROMS models.
